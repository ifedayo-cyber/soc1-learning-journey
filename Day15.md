Pyramid of Pain
Understanding the concept as a Threat Hunter, Incident Responder, or SOC Analyst is important.
What is Punycode? It is an attack used by attackers to redirect users to a malicious domain that seems legitimate at first glance. E.g ad1das.de == fake while adidas.de == real
Proxy logs or web server logs are used to detect malicious domains
To do hash lookups, use a tool like VirusTotal.
In the Pyramid of Pain, IP addresses are indicated with the color green.
                        Domain Names are indicated with the color teal
                        Host Artifacts belong to the yellow zone
